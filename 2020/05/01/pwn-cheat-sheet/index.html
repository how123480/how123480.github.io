May 01, 2020 in cheat sheet

GDB

run with argv

gdb —args {name} {argv[1], argv[2], …. }
ex:

x/nuf [address]

n: repeat count
u: unit size

u unit size (byte)
b 1
h 2
w 4
g 8

f: display format

f format
x hex
d int
s string
i instruction

otehr command

u unit size (byte)
return run until this function end
finish finish this process
clear clear the screen
q exit

set register

set ${register}=value
ex:

ascii tbl
